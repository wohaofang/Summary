## Ts 学习笔记

#### TypeScript介绍
```
TypeScript 扩展了JavaScript语法，任何已经存在的JavaScript程序，可以不加任何改动，在TypeScript环境下运行。

TypeScript只是向JavaScript添加了一些新的遵循ES6规范的语法，以及基于类的面向对象编程的这种特性。
```

- 编译 tsc class.ts => class.js
- basis =>  基础部分
- class => 类

### 运行ts
```
$ npm i -g ts-node

# 执行当前文件夹下 demo.ts 文件
$ ts-node demo.ts
# 输出的执行结果
```
